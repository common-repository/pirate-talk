Converts your normal blog posts to pirate talk on International Talk Like A Pirate Day (September 19th.)

== Description ==

Uses the pirate conversion tool found from http://developer.yahoo.net/blog/archives/2009/09/ahoy\_mates\_conv.html to convert all of your boring blog posts to pirate talk on talk like a pirate day! Arrrr!
For more infomation on this holiday visit http://www.talklikeapirate.com/
== Installation ==

= Automattic Install (recommended) =
1. Login to your Dashboard as an admin. 
2. Once logged in select Plugins > Add New from the main menu. 
3. Next search for "pirate-talk" (no quotes) and click the install button. 

= Manual Install =
1. Extract the contents of pirate-day.zip and upload to the wp-content/plugins folder.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy your blog talking like a pirate on September 19th


== Frequently Asked Questions == 

= I activated the plugin and nothing happend =
By default this plugin is only set to work on International Talk Like A Pirate Day which is September 19th. 
If you would like your blog to always talk like a pirate simply go to the the Pirate Day settings page and select "Always"  

= How does timezone support work =
For this feature to work you must set your timezone in your blogs wordpress general settings.
